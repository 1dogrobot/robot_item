# robot_item
本次主要是为项目机器人做自主路径规划，现阶段只是基于数据集的测试。

基于kitti数据集的使用，完成对雷达点云的二维标定和三维标定，imu的加入以及车体mesh的加入，navigation将会在后续发布。

在使用之前需要先用kitti2bag将数据集转化为ros可执行文件，以及点云和图片发布。
具体可参考我之前写的博客：
